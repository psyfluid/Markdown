# Руководство по языку разметки Markdown

## Выделение текста

*Курсив* или _Курсив_

**Полужирный** или __Полужирный__

Для комбинации полужирного и курсива используется сочетания разных символов: __*так*__ или _**так**_.

~~Зачёркнутый текст~~

___
## Списки

### Ненумерованные списки

* element 1
* element 2
* element 3

- element 1
- element 2
- element 3

+ element 1
+ element 2
+ element 3

### Нумерованные списки

1. element 1
2. element 2
3. element 3

### Вложенные списки

1. element 1
    1. element 1.1
    2. element 1.2
2. element 2
---
1. element 1
    * element 1.1
    * element 1.2
2. element 2
---
* element 1
    * element 1.1
    * element 1.2
* element 2
